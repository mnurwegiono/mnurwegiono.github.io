# Academic Website (GitHub Pages)

This is a responsive, multi-page academic website template for **Muhammad Nurwegiono**.

## Quick start
1. Upload the whole folder to a GitHub repository.
2. Go to **Settings â†’ Pages**.
3. Set:
   - Source: **Deploy from a branch**
   - Branch: `main` (or `master`)
   - Folder: `/ (root)`
4. Open the provided GitHub Pages URL.

## Update content
Edit:
- `assets/data.json` (publications, projects, talks, etc.)
- Each HTML page for narrative text

## Notes
- No build tools needed (pure HTML/CSS/JS).
- Animations: IntersectionObserver reveal effects.
